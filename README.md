# Toggle Character Case in Python

This Python program changes uppercase letters to lowercase and lowercase letters to uppercase in a string.

Explanation
The function toggleChar() is created to change the case of characters
The loop checks each character in the string
If the character is uppercase:
It converts it to lowercase
Otherwise:
It converts it to uppercase
The modified string is stored in the result
Finally, the toggled string is printed
